>[植物生长监测系统](Plant%20growth%20monitoring%20system.md)
>- 设计制作一个植物生长监测系统，通过Arduino控制传感器模块，实时监测植物的生长情况，并通过串口通信将数据传输到电脑端，通过串口监视器显示出来。
